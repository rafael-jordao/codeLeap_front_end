## Instructions

To run the App, use the script:

```bash
yarn dev
```
```bash
npm run dev
```

To compile the code into JavaScript, use the script:

```bash
yarn build
```
```bash
npm run build
```

Now we have a dist folder with the bundle of our project. To run it, run the command:

```bash
yarn preview
```

```bash
npm run preview
```
